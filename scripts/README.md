# Workflow Scripts

This folder keeps the historical project scripts organized by workflow step.

## Structure

- `step1/` trend analysis and spreadsheet export
- `step2/` multi-source validation
- `step3/` SKU selection
- `step4/` sourcing and supplier review
- `step5/` same-style candidate confirmation
- `step6/` material-pack generation, preview, and iteration scripts
- `tools/` local helper scripts

## Notes

- These scripts are kept for iteration and reproducibility.
- The plugin and skill flow is the primary user-facing entry.
- Step 6 contains a larger number of round-based scripts because that phase was iterated heavily.
