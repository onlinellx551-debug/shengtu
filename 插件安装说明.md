# 日本男装六步流插件安装说明

## 当前结构

这套方案是“Git 仓库 + 插件入口”的混合方式：

- 仓库源码位置：
  `C:\Users\Administrator\Desktop\Codex Project\task-codex\plugins\jp-menswear-workflow`
- 本地插件安装位置：
  `C:\Users\Administrator\plugins\jp-menswear-workflow`
- 本地 marketplace：
  `C:\Users\Administrator\.agents\plugins\marketplace.json`

## 安装 / 同步步骤

1. 运行同步脚本：

```powershell
powershell -ExecutionPolicy Bypass -File "C:\Users\Administrator\Desktop\Codex Project\task-codex\sync_jp_menswear_plugin.ps1"
```

2. 重开 Codex

3. 打开 `Plugins`

4. 检查是否出现：

- `JP Menswear 6-Step Workflow`

## 说明

- 仓库里的插件目录是源码
- 家目录里的插件目录是 Codex 读取的本地安装副本
- 每次更新插件源码后，都要再运行一次同步脚本
